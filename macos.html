Инструкция по настройке Zaborona VPN для MacOS
Данная инструкция подходит для MacOS 10.7.5 и новее
1
Установите программу Tunnelblick.
Рекомендуется устанавливать версию Beta. Для установки запустите файл из архива. На вопрос о конфигурационных файлах, нажмите «у меня есть конфигурационный файл»
2
Скачайте архив с файлом автоконфигурации VPN и распакуйте его.
https://zaborona.help/zaborona-help.tblk.zip
3
Двойным кликом по файлу настроек zaborona-help.tblk установите профиль Zaborona VPN.
Подключитесь к VPN.
Как удалить Tunnelblick с компьютера Mac?
Инструкция по удалению на официальном сайте:
https://www.tunnelblick.net/cUninstall.html

  1. Скачайте программу для удаления https://www.tunnelblick.net/release/Tunnelblick_Uninstaller_1.8_build_4405.dmg
  2. Отключитесь от VPN, запустите программу для удаления
